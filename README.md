# WORKTHESPACE - The Tactical Board for Real Life

A premium, high-performance landing page for a football team management app. Built with Next.js, React, and Tailwind CSS.

## Features

- **Interactive Tactical Boards**: Hover over the phone to reveal 4-4-2, 4-3-3, and 5-a-side formations.
- **Stadium Glow Aesthetics**: Custom animations and gradients for a premium sports broadcast feel.
- **Next.js App Router**: Optimized for performance and SEO.
- **Responsive Design**: Fully optimized for mobile and iOS devices.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: Custom CSS Keyframes

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/components/sections/hero.tsx`: The main interactive hero section.
- `src/app/globals.css`: Custom animation definitions.
- `src/components/layout/navbar.tsx`: Mobile-responsive navigation.
